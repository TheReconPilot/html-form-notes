HTML Form Tags - Notes

Self Notes, Plus a Codecademy Project

The purpose of a <form> is to allow users to input information and send it.

The <form>'s action attribute determines where the form's information goes.

The <form>'s method attribute determines how the information is sent and processed.

When a form is submitted, the name of the fields that accept input and the value of those fields are sent as name:value pairs.

The <input> element

We use the <input> element to add fields for users to input information.

We mention the different types using the type attribute.

Type Description
text Single row field for text input
password Single row field that censors text input
number Single row field for number input
range Slider to select from a range of numbers
checkbox Single checkbox which can be paired with other checkboxes
radio Radio button that can be paired with other radio buttons
list Pairs the <input> with a <datalist>
submit Creates the submit button

The <select> element

The <select> element is populated with <option> element and renders a dropdown list selection.

The <datalist> element

The <datalist> element is populated with <option> element and works with an <input> to search through choices.

The <textarea> element

The <textarea> element is a text input field that has a customizable area.
